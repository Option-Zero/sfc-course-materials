## Learning Objectives

Geospatial and AI are big topics! We're not going to master either of them in this 1-week module - instead we'll aim to get a broad overview, and then a deep dive into one specific intersection.

	- Become acquainted with:
	

		- how spatial data is represented

		- top climate-relevant sources of spatial data

		- how to use AI to scale spatial data insights

	

	

	- Map deforestation using spatial data and AI-driven land cover classification
